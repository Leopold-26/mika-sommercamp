# Mika für Founder einrichten

Der einfache Weg ist: GitHub öffnen, ZIP herunterladen, Ordner in Codex öffnen, `Starte Mika.` schreiben.

Du brauchst keine Terminal-Befehle.

## Schritt 1: GitHub öffnen

Öffne:

```text
https://github.com/Leopold-26/mika-sommercamp
```

## Schritt 2: ZIP herunterladen

Auf GitHub:

1. Klicke auf den grünen Button **Code**.
2. Klicke auf **Download ZIP**.
3. Entpacke die ZIP-Datei.

Danach liegt der Mika-Ordner auf deinem Laptop.

## Schritt 3: Ordner in Codex öffnen

Öffne den entpackten Mika-Ordner in Codex.

Wichtig: Öffne den ganzen Ordner, nicht nur eine einzelne Datei.

## Schritt 4: Mika starten

Starte in Codex einen neuen Chat in diesem Ordner und schreibe:

```text
Starte Mika.
```

Mika richtet dann den Sommercamp-Arbeitsmodus ein und beginnt mit dem Onboarding.

## Schritt 5: Unterlagen schicken

Ziehe vorhandene Materialien direkt in den Mika-Chat oder schicke Links.

Das können sein:

- Pitch Deck
- Notizen
- Screenshots
- Figma-Export
- Prototyp-Link
- Landing-Page-Text
- User Research
- bestehender Code

Mika liest zuerst, was schon da ist. Danach fragt Mika nur nach fehlenden Informationen.

## Was Mika danach macht

Mika führt dich durch diese Reihenfolge:

1. aktuellen Stand prüfen
2. offene Fragen stellen
3. Produktidee und Zielnutzer schärfen
4. Distribution und erste 100 Nutzer prüfen
5. 10-Wochen-Plan mit Launch in Woche 4 erstellen
6. Lücken im Plan mit dir besprechen
7. Sprint 1 starten
8. Website, Landing Page oder App-MVP bauen, wenn der Plan klar genug ist
9. QA, Launch, Retros und Updates begleiten

## Wenn du schon ein eigenes Projekt hast

Starte trotzdem zuerst im Mika-Ordner.

Du kannst Mika dann deine vorhandenen Dateien, Links oder dein bestehendes Repo im Chat geben. Mika hilft dir, den aktuellen Stand zu prüfen und den nächsten sinnvollen Schritt zu planen.

## Advanced: Plugin installieren

Nur wenn du Mika dauerhaft als Codex-Plugin installieren willst, nutze Terminal:

```bash
codex plugin marketplace add Leopold-26/mika-sommercamp --ref main
codex plugin add mika-sommercamp@gruenderszene-sommercamp
```

Danach Codex neu starten.

Für die meisten Founder ist dieser Advanced-Weg nicht nötig.
