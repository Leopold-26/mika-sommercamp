# Uploads Für Mika

Lege hier alles ab, was du schon hast.

Mika soll diese Dateien zuerst lesen, bevor neue Fragen gestellt werden. Ziel ist, dass du deine Arbeit nicht doppelt machst.

Gute Uploads sind:

- Pitch Decks
- Notion/Google Docs Exporte
- PDFs
- Markdown- oder Textdateien
- Screenshots
- Figma-Exports
- Landing-Page-Texte
- User-Interviews
- Wettbewerbsrecherche
- bestehender Code
- technische Skizzen
- offene Fragen

Wenn ein wichtiges Material nicht als Datei vorliegt, fuege in einer Textdatei den Link und eine kurze Beschreibung ein.

Beispiel:

```text
prototype-link.txt

Link: https://...
Was es ist: Erster klickbarer Prototyp.
Was Mika prüfen soll: Ob der erste Nutzer den Aha-Moment versteht.
```
