# Mika Setup für Founder

## Empfohlener Weg: Download ZIP

1. Öffne `https://github.com/Leopold-26/mika-sommercamp`.
2. Klicke auf **Code**.
3. Klicke auf **Download ZIP**.
4. Entpacke die ZIP-Datei.
5. Öffne den entpackten Ordner in Codex.
6. Schreibe im neuen Codex-Chat:

```text
Starte Mika.
```

Das reicht. Mika funktioniert in diesem Ordner, weil die Sommercamp-Anweisungen und Skills schon enthalten sind.

## Unterlagen geben

Ziehe vorhandene Unterlagen direkt in den Mika-Chat oder schicke Links:

- Pitch Decks
- Notizen
- Screenshots
- Figma-Exports
- Prototyp-Links
- Landing-Page-Texte
- User Research
- bestehender Code

Mika analysiert zuerst, was schon da ist, und fragt danach nur nach fehlenden Informationen.

## Optional: Plugin-Weg

Der Plugin-Weg ist nur für Personen gedacht, die Mika dauerhaft in mehreren Codex-Projekten nutzen wollen.

```bash
codex plugin marketplace add Leopold-26/mika-sommercamp --ref main
codex plugin add mika-sommercamp@gruenderszene-sommercamp
```

Danach Codex neu starten.

Für Founder im Sommercamp ist **Download ZIP** einfacher.
